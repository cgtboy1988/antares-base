Attribute ( antares.attribute.Attribute )¶

class antares.attribute.Attribute(name, atype, context, datatype, scale, description='')¶

This is the Attribute class. An attribute is initialized with its name, the context it belongs to, the datatype of its value, a scale, and the description of its value.

Param:name(string): name of the attribute
Param:context(antares.context.Context): context the attribute belongs to
Param:datatype(string): data type of the attribute value
Param:scale(IntPair or FloatPair): the range to which the attribute value scales
Param:scale(FloatPair): the range to which the attribute value scales
Param:description(string): description of the attribute
